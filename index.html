Introdução

Resumo: SDN ( Software Definied Networking ) é uma arquitetura emergente, que tem como principais objetivos ser dinâmica, adaptável, permitir que os administradores da rede gerenciem os seus serviços a partir de uma abstração de alto nível das funcionalidades e possuir uma relação custo benefício boa, tornando-se ideal no cenário das aplicações atuais, que são dinâmicas e necessitam grande banda de rede. A principal característica desta rede é a separação entre o sistema que decide para onde o tráfego é enviado e os sistemas que direcionam o tráfego para o destino selecionado.
